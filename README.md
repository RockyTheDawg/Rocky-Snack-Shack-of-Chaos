# Rocky FunHouse

A furry, food-themed easter egg page starring Rocky, an Aussie shepherd with a very serious snack agenda.

## Uploading To GitHub

Upload these files to the main/root area of your repository:

- `index.html`
- `styles.css`
- `script.js`
- `README.md`
- `aussie-tab-logo-transparent-512.png`

Keep your existing `assets` folder too. The page uses these existing asset files:

- `assets/rocky-puppy.png`
- `assets/funhouse-background.png`
- `assets/dog-treat.png`
- `assets/river-cat.png`

Do not upload the `outputs` folder itself.
